# Démo

## Une fois le README du vue effectuer et celui-ci aussi. tous dois être lancé.

## Aller sur le [localhost](localhost:5173)

## Pour ajouter des pièces

- aller sur le [/admin](localhost:5173/admin)
- MDP: admin
- Cliquer sur "Ajouter une pièce"
- Tous les champs sont obligatoire
- enfin cliquer sur ajouter

## Pour supprimer une pèce

- toujours dans le mode admin
- cliquer pour suprimer une pièce
- ATTENTION il n'y a pas de retour possible une fois cliquer

## Pour modifier une pièce

- toujours dans le mode admin
- cliquer sur modifier
- puis renseigner els champs
- et modifier

## Pour récupérer les pièces

- quitté le mode admin
- vous aurez toutes les pièces ajoutées au choix

## Pour commander
- toujours dans le mode client(non admin)
- choisir les pièces
- aller dans le pannier
- passer commande
